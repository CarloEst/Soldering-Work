Interactive Panda

Creative Embedded Systems - Spring 2026

Short Description

An interactive panda built with creative embedded systems.

Reflection

In this project, the goal was to create an interactive panda that would move and interact with the user using an ESP32. I wanted to have a stealth approach where the components and wires would be hidden inside the panda, but that was proven hard to do. Instead, I opted for approach where the wires and the copper tape would be part of the aesthetic of the panda. I plaed the copper tape in ways that present an augmented reality effect, and I decided to have the wires go through the panda' back side to give to give this idea of an exoskeleton that augment the panda's natural movement.
