# ESP32 - Generative Art
This is an ESP32 with a TFT Display. We're displaying a generative art of showing a sea waves, a mountain, and the caribbean sun that reflects the English translated lyrics of the Haitian song "Ayiti se" by Mikaben.


These are specifics of the esp32 we're using:
- Model: LilyGo T-Display PICOC3
- Xtensa® single-/dual-core 32-bit LX6 microprocessor(s)
- 448 KBROM
- 520 KBSRAM
- Added 3.7V Battery (Not included with ESP32)
